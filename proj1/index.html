Project 1: Images of the Russian Empire

Project Overview

This project focuses on reconstructing color photographs from the Prokudin-Gorskii collection. The collection contains three separate black-and-white images of a scene, each through a different color filter: red, green, and blue. The primary challenge of this project is to take these three individual color channel images and digitally align them. By correctly layering the red, green, and blue channels, we can recreate the full-color photograph.

Part 1: Single-Scale Alignment

My Approach

For the single-scale alignment, I implemented an exhaustive search method. This approach tests all possible pixel displacements within a defined window (e.g., [-15, 15] pixels) for both horizontal and vertical shifts. To determine the best alignment, I tried two image similarity metrics: the L2 Norm (Sum of Squared Differences) and Normalized Cross-Correlation (NCC). Both metrics were effective in identifying optimal alignments for the lower-resolution images.

A key challenge encountered during development was the presence of varying borders in the input images, which negatively impacted the accuracy of the similarity calculations. To mitigate this, I implemented a cropping strategy, removing 6% from each edge of the images (resulting in a 12% reduction in both height and width). This step improved alignment quality and computational efficiency.

All images below were produced using the NCC metric and single-scale search.

Part 2: Multi-Scale Pyramid Alignment

My Approach

To overcome the computational cost of exhaustive search on high-resolution images, I implemented a recursive image pyramid alignment algorithm. This approach works by creating a series of progressively smaller versions of the input images.

The process begins by downscaling the images by a factor of two. The function is then called recursively on these smaller images. Once an optimal displacement is found at a lower resolution, this displacement is scaled up and used as an initial estimate for a more refined search on the next higher resolution level (or the original image). The refined search is done using the simple-scale method. This significantly reduces the search space at each level, making the alignment process much faster and more efficient for large images. The base case for the recursion is when the image is small enough to be efficiently processed by the single-scale search.

The cropping strategy (6% from each edge) mentioned in the single-scale section was also applied here to ensure accurate alignment across all pyramid levels. I found NCC to perform better than the L2 Norm for the larger images.

All images below were produced using the NCC metric and multi-scale search.
